### 기능

- 글 목록 조회
- 글 상세 조회
- 글 작성 (파일 등록 가능) / 수정 / 삭제
- 댓글 등록
- 제목/내용/작성자 로 글 검색

## 🛠 사용 기술 및 라이브러리

- Java
- Spring
- Oracle
- JavaScript
